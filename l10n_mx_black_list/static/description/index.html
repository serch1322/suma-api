Acciones Automáticas

Black List

Se crea una acción planificada para actualizar la lista de contribuyentes.

Funcionalidad

Restringir creación de Contactos

Cuando se crea un nuevo contacto, se verifica que el RFC de este no se encuentre presente en la lista negra; en caso de ser así, se bloquea la posibilidad de crearlo.

Restringir creación de Facturas

De manera analoga a la creación de contactos, cuando se crea una nueva factura también se comprueba que el Cliente no forme parte de la lista; esto con la finalidad de detectar clientes que anteriormente no estaban en dicha situación.
